# API Project

This is a simple API project built with Go.

## Getting Started

### Prerequisites

- Go installed on your system
- Make utility

### Running the Project

To run the project, simply use:

```bash
make run
```

## Documentation

API documentation is available at `/docs` endpoint when the server is running.
